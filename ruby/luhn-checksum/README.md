# luhn-checksum

The program checks the validity of an input card number using the [Luhn algorithm](http://en.wikipedia.org/wiki/Luhn_algorithm)

Just run

    $ ruby luhn-checksum.rb
    Card number: 4561261212345467
    Card number is OK
